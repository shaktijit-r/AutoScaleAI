# AutoScaleAI
AutoScaleAI: An Intelligent Auto-Scaling Framework for Microservices using Reinforcement Learning

## Quickstart


1. Create a Python virtualenv


```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```


2. Run the trainer (placeholder)


```bash
./scripts/run_local.sh
```


3. Configure Prometheus and Kubernetes access for end-to-end testing.
